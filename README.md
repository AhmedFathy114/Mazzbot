# 💊 Mazzbot

**Mazzbot** is a modern healthcare web application designed to help users manage their medications, build healthy habits, and stay committed to their treatment plans.

The application provides a simple and user-friendly experience for tracking medicines, monitoring daily health activities, generating medical reports, and storing personal medical information securely.

## ✨ Features

- 🔐 Secure Authentication (Email & Google) using Supabase
- 👤 Personal Medical Profile
- 💊 Medication Management (Add, Edit, Delete, Track)
- ⏰ Smart Medication Reminders
- 📊 Daily Medication Adherence Tracking
- 💧 Water Intake Tracker
- 🎯 Healthy Daily Goals
- 📄 Medical Report Generation (PDF)
- 🚨 Emergency Contact Information
- 🌙 Dark & Light Theme
- 📱 Responsive Modern UI
- ⚡ Single Page Application (SPA) Experience

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Supabase
- jsPDF
- Chart.js

## 🎯 Project Goal

Our goal is to improve medication adherence by providing an easy-to-use healthcare assistant that helps users organize their medications, monitor their health, and access important medical information in one place.
